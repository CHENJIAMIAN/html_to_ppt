Vitest的安装和配置

详细介绍如何安装Vitest以及基本配置方法

download

安装Vitest

  • 1
    环境要求
    Node.js 18.0.0 或更高版本
  • 2
    安装命令
    npm install -D vitest
    或使用其他包管理器：
    yarn add -D vitest
    pnpm add -D vitest
  • 3
    添加测试脚本
    在package.json中添加：
    "scripts": {
      "test": "vitest"
    }
settings

配置Vitest

Vitest可以使用vite.config.js中的配置，也可以创建单独的vitest.config.js文件
import { defineConfig } from 'vite'

export default defineConfig({
  test: {
    // 测试文件匹配模式
    include: [
      '**/*.{test,spec}.{js,ts,jsx,tsx}'
    ],
    // 测试环境
    environment: 'jsdom',
    // 覆盖率配置
    coverage: {
      reporter: ['text', 'json', 'html']
    }
  }
})
environment
测试环境：'node'、'jsdom'、'happy-dom'
globals
启用全局API，无需导入
setupFiles
测试前运行的设置文件
reporters
测试报告输出格式
