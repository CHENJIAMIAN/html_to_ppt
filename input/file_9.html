模拟(Mocking)

Vitest提供vi辅助工具，支持函数、模块和定时器的模拟，便于隔离测试依赖

import { vi, test, expect } from 'vitest'

// 模拟函数
const mockFn = vi.fn();
mockFn('hello');
expect(mockFn).toHaveBeenCalledWith('hello');

// 模拟模块
vi.mock('./axios', () => ({
  default: {
    get: vi.fn().mockResolvedValue({ data: 'test' })
  }
}));
函数模拟 模块模拟 定时器模拟

快照测试

捕获组件输出的快照，用于跟踪UI变化，确保组件渲染结果符合预期

import { render } from '@testing-library/vue'
import { test } from 'vitest'
import MyComponent from './MyComponent.vue'

test('renders correctly', () => {
  const { container } = render(MyComponent);
  expect(container).toMatchSnapshot();
});

// 内联快照
expect(user).toMatchInlineSnapshot({
  name: 'John',
  age: 30
});
UI测试 变更检测 内联快照

代码覆盖率

内置代码覆盖率分析工具，支持多种报告格式，帮助评估测试完整性

// 配置覆盖率
export default defineConfig({
  test: {
    coverage: {
      reporter: ['text', 'json', 'html'],
      include: ['src/**/*'],
      exclude: ['node_modules/**'],
      thresholds: {
        global: {
          branches: 80,
          functions: 80,
          lines: 80,
          statements: 80
        }
      }
    }
  }
});
覆盖率报告 阈值设置 HTML可视化

性能优化

提供并行测试、智能缓存和选择性测试等特性，优化大型项目的测试执行效率

// 并行测试配置
export default defineConfig({
  test: {
    pool: 'threads', // 使用线程池
    poolOptions: {
      threads: {
        maxThreads: 4, // 最大线程数
        minThreads: 2 // 最小线程数
      }
    },
    maxConcurrency: 5 // 最大并发数
  }
});

// 运行特定测试
npx vitest --run --reporter=verbose fileName.test.js
并行执行 智能缓存 选择性测试
