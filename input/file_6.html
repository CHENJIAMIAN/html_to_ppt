Vitest的基本概念和特点

了解Vitest的核心概念、主要特点和优势

lightbulb 核心概念

Vitest是由Vite驱动的下一代测试框架，利用Vite的转换流水线，提供与开发环境一致的测试体验

speed

极速性能

利用Vite的快速转换和热重载能力，提供闪电般的测试执行速度

快速启动 并行执行 智能缓存
sync

智能监听模式

只重新运行相关更改的测试，提供即时反馈，类似热模块重载的测试体验

增量测试 即时反馈 开发友好
integration_instructions

Jest兼容性

提供与Jest兼容的API，支持Expect断言、快照测试、覆盖率等功能，迁移成本低

熟悉API 生态兼容 平滑迁移
settings

简单配置

与应用共享相同的配置(vite.config.js)，无需维护两套配置，降低维护成本

统一配置 开箱即用 插件生态
