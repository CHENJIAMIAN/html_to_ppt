课程练习

通过实践巩固Vitest测试工具的使用方法

1

functions 基础测试用例编写

为以下工具函数编写测试用例，验证各种边界条件和异常情况

•
为数组工具函数编写测试：filter、map、reduce
•
为字符串工具函数编写测试：trim、split、replace
•
使用describe组织测试套件，使用test定义测试用例
lightbulb
提示：考虑使用不同的断言方法（toBe、toEqual、toBeTruthy等）验证结果
2

sync 异步代码测试

测试异步操作，包括Promise、async/await和定时器功能

•
为API请求函数编写测试，使用vi.mock模拟API响应
•
测试定时器函数，使用vi.useFakeTimers模拟时间流逝
•
验证错误处理逻辑，确保异常情况被正确处理
lightbulb
提示：使用async/await语法处理异步测试，使用rejects/throws验证错误
3

widgets 组件测试实践

为Vue或React组件编写测试，验证组件渲染、交互和状态变化

•
测试表单组件的输入验证和提交功能
•
测试列表组件的数据渲染和交互功能
•
使用快照测试验证组件UI的一致性
lightbulb
提示：使用相应的测试库（Vue Test Utils或React Testing Library）辅助组件测试
